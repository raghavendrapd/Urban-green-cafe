# Urban Greens Cafe

A modern, responsive homepage for a local caf√©, built with React, Vite, and Tailwind CSS.

## Features

- **Hero Section**: Immersive background with call-to-action.
- **About Section**: Story and opening hours.
- **Menu Gallery**: Interactive tabs for categories (Drinks, Bakery, Specials).
- **Photo Gallery**: Grid layout with hover effects.
- **Testimonials**: Customer reviews.
- **Contact**: Functional-looking form and Google Maps embed.
- **Dark Mode**: Toggle between light and dark themes.
- **Responsive Design**: Optimized for mobile and desktop.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (ready for use)
